## Modular architecture additions this session

These exist now and any future work should reuse them:

### `ONBOARDING_PANELS` + `isOnboardingActive()` (~line 3982-3993)
Single source of truth for the unskippable first-play panels. Used by:
- Escape handler
- `openHelp()` (don't stack on top)
- touchstart double-tap detection (don't open rotor over tutorial)

To add a new onboarding panel: add its ID to the array. Every guard updates automatically.

### `closePanel(panelId)` (~line 3997-4003)
Modular replacement for inline `panel.classList.remove('show')`. Removes the class AND speaks "Closed." for blind feedback. **All 13 close-button handlers now route through this.**

### `PANEL_ROTORS` map (~line 4607-4621)
Object keyed by panel ID, value is a function returning rotor options. `getActiveRotorOptions()` does one map lookup. To add rotor coverage for a new panel:
```js
PANEL_ROTORS.myNewPanel = () => [
  { label:'My action', open: () => doSomething() },
  { label:'Close my new panel', open: () => closePanel('myNewPanel') },
];
```

### Toggle speak (modular, ~line 5007-5014)
Settings toggles all use the same speak() pattern — derives label from `.toggle-row .lbl` text. Adding a new toggle in the array (line 4994) auto-gets George narration if its label exists in the corpus, otherwise silent (no system voice).

## Gotchas / decisions worth remembering

1. **Manifest field naming.** The manifest uses `{ "text": "...", "wav": "audio/voice/foo.mp3" }`. The field is named `wav` for historical reasons; `decodeAudioData()` is format-agnostic. Don't rename — would break older entries.

2. **ElevenLabs free tier limits output formats.** PCM is gated to Pro. Use `mp3_44100_128` only. See `gen_*.py` scripts for the working incantation.

3. **API key in source files.** The four `gen_*.py` scripts have the key inlined. Sam supplied it directly; not committed to a public repo per current setup. Move to env var if deploying these scripts elsewhere.

4. **`silentIfUnmatched: true` is the standard pattern** for any speak() call with template literals or dynamic content. Without it, a corpus miss falls through to system "lady voice" — Sam's #1 reported pain point.

5. **Tests use `?fresh=1&nogate=1`.** `nogate` only bypasses welcome-speech gate; the unskippable tutorial gate (Batch 2) still applies. Tests that click `tutorialDoneBtn` rely on Playwright's auto-wait for `disabled=false` (~23s).

6. **The rotor is panel-aware.** Opening rotor while a panel is showing returns that panel's rotor (PANEL_ROTORS map). This is how Sam reaches "Close X" or panel-specific actions via gesture. Onboarding panels (welcome/intro/tutorial) explicitly block rotor during first play.

7. **Build-tag bumping is the iOS Safari cache buster.** Whenever you edit index.html, bump `ECHO_BUILD` (line 10). The manifest fetch URL (`?v=...`) and voice clip URLs (`?v=...`) pin to this. iOS Safari aggressively caches HTML; without the bump Sam sees stale builds.

8. **Co-authored-by tag.** No commits this session — droplet is not a git repo. Files are edited in place.
